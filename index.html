CSS Animation

CSS animation is a way for an element to transition from one style to another seamlessly.

Animating in CSS has two components:

  1. Style that describes the CSS Animation
  2. Keyframes that indicate the start and end of the animation

Here is how I animated my cat image:

  1. I added the image of a cat to the img tag.
  2. I assigned my image the class of “rotate”
  3. In my style.css file, I put “@keyframes” and bound the name “spin” to it.
  4. Within the “@keyframes” element, I coded “from {transform: rotate(0deg)” and “to {transform: rotate(360deg)” to ensure that the image would turn a full 360 degrees.
  5. In a different section I referenced the “rotate” class with the “:hover” attribute because I wanted the animation to occur only when someone is hovering over the element.
  6. Within the “.rotate:hover" section I added “animation: spin 2s infinite;” to tell the code to allow the rotating to happen for two seconds per spin and to do it infinitely without stopping.

    7. And VOILA!
